Welcome to the "Abisso dei Draghi Volanti" survey

This repository contains the survey data of the cave.

All the data are provided under a Creative Commons Attribute-NonCommercial-ShareAlike licence.

How to create the cave maps

  • download the data zip or tar.gz
  • uncompress it on your PC
  • have Therion (therion.speleo.sk) installed
  • execute therion in the project root directory
  • the cave map pdf's are saved in the project root directory
